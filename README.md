# MultiIB

### Build and run
- install python
- create an Environment:

	`py -3 -m venv .venv`

- before you work on your project, activate the corresponding environment:

	`.venv\Scripts\activate`

- install Flask:

	`pip install flask`

- install ib_insync:

	`pip install ib_insync`

- to run the app:

	`flask --app main run`
